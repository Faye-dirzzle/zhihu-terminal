## 项目介绍
本项目为知乎的终端版实现,基于asyncio实现全异步操作

运行此项目即可以使用命令行的方式来操作知乎,功能涵盖:浏览,点赞,感谢等功能,未来将实现知乎网页端的绝大部分功能

## 运行环境
Python 3.7

项目在Mac OSX 10.14.5 进行开发,目前未进行windows系统的适配

## 准备

安装Python 3.7的环境后执行
```
pip install -r requirements.txt
```
